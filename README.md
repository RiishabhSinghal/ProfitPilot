# ProfitPilot

ProfitPilot is an AI-powered business analytics platform that identifies profitable customers, analyzes transactional datasets, and generates business insights using LLM-powered agents.

## Features

- Excel/CSV Data Loading
- Dataset Validation
- Automatic Schema Extraction
- Data Quality Auditing
- Shared Business State Management

## Project Structure

- ExcelEngine
- AuditEngine
- BusinessState
- Custom Exceptions
- Logging

## Tech Stack

- Python
- Pandas
- Ollama (Planned)
- LangGraph (Planned)
- Streamlit (Planned)

## Roadmap

- Multi-Agent Architecture
- Customer Segmentation
- Profitability Engine
- Interactive Dashboard
- Automated Business Reports
